# STL Toolkit (C++)

A small C++ project demonstrating practical usage of the C++ Standard Template Library (STL).

This repository is created as part of my GSoC preparation to practice:
- STL containers
- STL algorithms
- Clean C++ project structure
- Git & GitHub workflow

## Features
- Frequency counting using maps
- Sorting utilities using STL algorithms
- Set operations using STL sets
- Vector traversing using iterators
- Key-value lookup and ordered traversal using STL map

## How to Compile and Run

```bash
g++ src/frequency.cpp -o freq
./freq
